# 🛡️ Security Policy

## Reporting Security Vulnerabilities

If you discover a critical security vulnerability, please **do not** open an issue on GitHub.

Instead, **report the vulnerability privately** via the <kbd>Security</kbd> tab so I have an opportunity to fix the issue before it is publicly disclosed. Thanks :)

**Link:** Open the <kbd>Security</kbd> tab for the project you're using.
